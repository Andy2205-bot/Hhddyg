Employee clocking system project description


I want to develop an employee clocking system where the system will allow the employee to clock_in in the morning and clock_out when they leave to lunch. And the employee should clock_in again after lunch and at the end of the day they clock out. The system should also allow the administrator to view the employee worked hours. The system should allow the admin to calculate the worked hours of every employee per day and prints the weekly report.
